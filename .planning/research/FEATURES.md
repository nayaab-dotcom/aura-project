# Research: Features dimension for AURA

## Features for Disaster Response Dashboards

### Table Stakes (Basics)

- **Real-Time GPS Mapping**: Essential for tracking drones and identifying survivor locations.
- **Sensor Telemetry Display**: Live battery, signal, and environmental (temp/gas) monitoring.
- **Mission Overlays**: Heatmaps showing risk zones (High/Medium/Safe).
- **Incident Logs**: Permanent record of detected hazards and survivor finds.

### Differentiators (Strategic Advantages)

- **Swarm Orchestration**: Coordinated search patterns (lawnmower, spiral) for faster coverage.
- **Dynamic Pathfinding (A*)**: Automatic, safe return-to-base paths avoiding High-risk zones.
- **Differential Imaging**: Comparing pre-disaster maps with live drone footage (simulated).
- **Automated Survivor Detection**: Algorithm-driven probability markers for survivors.

### Anti-Features (What to Skip for Now)

- **Real Hardware Integration**: Stay with simulated sensors (as per PROJECT.md).
- **Multi-user Login**: Unnecessary complexity for a demo.
- **Cloud Analytics**: Local processing is sufficient (and potentially faster).

## Confidence Level: High
These features are industry standard for crisis management toolings as of 2025.
